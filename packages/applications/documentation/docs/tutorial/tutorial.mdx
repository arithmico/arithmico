# Einstieg

Arithmico ist eine barrierefreie Software zum Berechnen von mathematischen Ausdrücken.
Mit Arithmico kannst Du alle gängigen Funktionen eines wissenschaftlichen Taschenrechners (WTR) nutzen.

Wenn Du Arithmico nutzen möchtest, kannst du einfach unsere Onlineversion [hier](https://arithmico.com) nutzen. Wenn Du eine Offlineversion für eine Klausur benötigst, kannst Du dir eine passende in unserem [Downloadportal](https://blog.arithmico.com/downloads) herunterladen.
